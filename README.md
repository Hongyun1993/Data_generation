# Data_generation
The data_generation_system, use the MaskRCNN and graphCut method to generate the comp_images.

## The Dependency
1. Opencv-python 3.1.0
2. Numpy 1.14.5
3. TensorFlow 1.3.0
4. Keras 2.0.8 
5. Scipy
6. Pillow
7. Scikit-image
